# Sepsis-Early-Detection-using-Machine-Learning-on-Real-ICU-Patient-Data
Built an end-to-end early sepsis prediction system using real ICU patient time-series data (~1.5M records). Implemented leakage-safe patient-wise preprocessing, temporal feature engineering, and imbalance-aware XGBoost modeling. Achieved 0.825 ROC-AUC with 68% sepsis recall for early critical care risk detection.
